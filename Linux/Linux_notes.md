paramètre court : un tiret + une seule lettre 

#### Exemple :
`commande -d`

#### Plusieurs paramètres : 
`commande -d -a -U -h`

ou :

`commande -daUh`


paramètre long :  deux tirets + plusieurs lettres

#### Exemple :
`commande --parametre`

#### Plusieurs paramètres :
`commande --parametre1 --parametre2`


Il est également possible de combiner les paramètres courts/longs : `commande -daUh --autreparametre` .
