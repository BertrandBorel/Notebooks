# <center> Helix commandes </center>

## ouvrir Helix : 
`hx`

## commandes :
`echap:commande` <br>

**quitter :** `echap:quit`

## déplacements :
- haut : `K`
- bas : `J`
- => `L`
- <= `H`

### aller tout en haut :
`G`
### aller à une ligne :
`G` + ligne
### aller tout en bas :
`E`

## mode insertion :
`i`
